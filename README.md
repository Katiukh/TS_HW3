# Time Series Forecasting (M4)

В работе исследуются методы прогнозирования временных рядов на данных M4 (100 случайных рядов).

## Гипотеза
Цель работы — определить, какой способ моделирования сезонности наиболее эффективен.

Для этого:
- использован датасет M4 с несколькими временными рядами, обладающими выраженной сезонностью (подтверждено ACF)
- зафиксированы бейзлайны (Naive, Seasonal Naive, AutoETS, AutoTheta) и глобальная ML-модель
- сравнены различные наборы признаков:
  - только лаги
  - лаги + сезонные лаги
  - лаги + календарные признаки
  - лаги + Фурье-признаки
  - комбинации признаков
- исследовано влияние горизонта прогнозирования и структуры рядов на качество модели

## Что сделано
- Проведен EDA 
- Построены модели-бейзлайны
- Реализован ML-подход Direct MIMO с различными признаками
- Проведен подбор конфигураций лагов и горизонта 

## Метрики
- sMAPE
- MASE

## Результаты
- Базовые модели (AutoETS, AutoTheta) показали лучшие результаты по MASE (MASE < 1)
- ML-подход не превзошел бейзлайны по MASE
- ML-подход превзошел бейзлайны по sMAPE
- Добавление сезонных лагов улучшает качество
- Более сложные признаки (календарные, Фурье) не дали устойчивого улучшения
